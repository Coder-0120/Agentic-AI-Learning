# 🤖 AI Agent Learning Journey

This repository documents my journey of learning and building AI agents from scratch. It contains hands-on experiments, code examples, and implementations of core AI agent concepts using modern AI models and tools.

## 🚀 What I Learned

### 📌 AI Agent Fundamentals

* Setting up and interacting with AI models
* Understanding request-response workflows
* Building the first "Hello World" AI Agent

### 💬 Multi-Turn Conversations

* Sending messages to AI models
* Managing conversation history
* Generating context-aware responses

### ⚡ Streaming Responses

* Receiving responses in real time
* Handling streamed output efficiently
* Improving user experience with live generation

### 🔍 Tool Integration

* Connecting AI agents with external tools
* Integrating Google Search capabilities
* Enhancing agent responses with real-time information

### 📊 Response Metadata

* Extracting metadata from AI responses
* Understanding token usage and model outputs
* Processing structured response data

## 🛠️ Technologies Used

* JavaScript / Node.js
* Gemini API
* Google Search Tools
* Agentic AI Concepts

## 🎯 Goal

To understand the fundamentals of Agentic AI and progressively build intelligent agents capable of reasoning, using tools, handling conversations, and interacting with real-world data.
